echo 和 print 区别:

  • echo - 可以输出一个或多个字符串
  • print - 只允许输出一个字符串，返回值总为 1

提示：echo 输出的速度比 print 快， echo 没有返回值，print有返回值1。(php与C语言有相同之处)

Index.php

index是普遍意义上的“首页”，也就是你输入一个域名后会打开一个页面，基本上就是index.xxxx（基本上首页都不会把index.xxxx显示在url里，但也不绝对）

后面的php是“Hypertext Preprocessor”，一个脚本语言，与asp、jsp一样是用来处理网站各种事物的程序。

语句

双引号里的语句变量能够被识别；

单引号里的语句变量不能够被识别；

example：$a=’1’;

​ echo “123 $a”;——浏览器打印为123 1

​ echo “123 $a”;——浏览器打印为123 $a

var_dump($a, $b, $c, 123, 2.2);

调试使用，可输出数据类型 字符串连接且加空格‘ ’

$a = (string)1;

数值左边括号里为想要强行转换的数据类型，在这过程中可能会有损失。

危险函数

一、PHP代码执行函数

mixed eval(string $code)

把字符串$code 作为PHP代码执行

很多常见的webshell都是用eval来执行具体操作的

函数 eval() 语言结构是 非常危险的， 因为它允许执行任意 PHP 代码。
