二维数组（矩阵）

例子：int a[6] [6] ;//6*6,6行6列

访问二维数组

例子: a[0] [0] 访问a数组，第一行第一列中的元素

注意防止越界访问（取值范围）

二维数组也可以对部分元素赋值

如 int a[3] [4]="{{1}，{5}，{9}}"每一列第一个数
